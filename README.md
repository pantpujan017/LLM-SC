LLM-SC: Pashupatinath Temple Experience Analysis
Overview
An expanded version of the LLM-SC Fusion Model (LLM + Social Computing) applied to Pashupatinath Temple, Kathmandu.
This project analyzes thousands of tourist reviews using smart prompt engineering to understand how visitors truly experience this sacred UNESCO site — from spiritual feelings to practical issues.
What We're Doing

Extracting detailed aspects, emotions, and relationships from reviews
Building rich semantic networks to see the bigger picture
Comparing results with traditional methods

Current Status
Actively working on it (May 2026).
Data collection is done. Right now focused on fine-tuning prompts specifically for Pashupatinath’s unique religious and cultural context.
Goal: Help improve tourist experience while supporting sustainable preservation of this living heritage site.
